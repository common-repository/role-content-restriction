Restricts access to selected post types based on user roles.



== Description ==



This plugin restricts access to default and custom post types based on user roles. The plugin displays a custom message to those who do not have access to view the content.





== Installation ==



1. Upload 'role-content-restriction' to the '/wp-content/plugins/' directory

2. Activate the plugin through the 'Plugins' menu in WordPress

3. Select restriction options when creating a post or a page.

== Frequently asked questions ==



Q: Can I restrict only a single page or post?

A: Yes, you can add restirction to individual pages, posts or custom post types.
